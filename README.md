# 📸 Kartwod_Photoo - Photography Portfolio & CMS Mockup
Welcome to the **Kartwod_Photoo** repository! This project is a fully static, responsive website built exclusively with HTML5 and CSS3 (Zero JavaScript). It serves as a comprehensive portfolio for a photographer, coupled with a fully stylized mock-up of an administrative backend panel.

### 🎓 Project Context
This project was developed as a final assignment for the "Web Design" (*Projektowanie serwisów internetowych*) course. 
It strictly adheres to **W3C** and **WCAG** validation standards and is designed to remain readable even if CSS is disabled.

### 🌟 Key Features
The project is divided into two main environments, simulating a complete web service using only CSS logic (e.g., `:target`, `:hover`, `@keyframes`):

#### 🖥️ Frontend (Client-Side)
* **Home:** Welcoming interface with service descriptions and a pricing table.
* **Marketplace (Giełda):** A board for photography equipment ads, utilizing CSS-only expandable content.
* **Gallery:** A responsive photo grid with zoom-on-hover effects.
* **Contact:** A fully styled contact form layout.
* **Authentication:** Login and Registration form layouts.
* **Info:** Student details, browser testing context, and validation badges.

#### ⚙️ Backend (Admin Panel Mockup)
* **My Profile:** Data visualization and a mock-up of a profile editing form.
* **Posts & Users:** Interactive data tables with simulated "Edit" and "Delete" modal windows handled entirely via CSS.
* **Statistics:** A dashboard featuring analytical tiles (Visits, Session Time, Bounce Rate, etc.).

---

### 🎨 UI/UX & Design System
* **Theme:** Modern, dark-themed aesthetic with a static background image.
* **Elements:** Semi-transparent containers, backdrop filters (blur), and light, thin borders.
* **Color Palette:** Deep blacks, crisp whites, and vibrant light blue accents for high contrast.
* **Animations:** CSS-driven decorative elements (e.g., falling camera illustrations) and smooth transitions.

---

### 🛠️ Technologies Built With
* **HTML5** (UTF-8 encoding, semantic structure)
* **CSS3** (Flexbox, CSS Grid, Media Queries for Responsive Web Design)
* *Note: No JavaScript or PHP was used. All interactivity is purely CSS-driven.*
